# Formation of wide-orbit giant planets in protoplanetary disks with a decreasing pebble flux
Code for figures in the paper "Formation of wide-orbit giant planets in protoplanetary disks  with a decreasing pebble flux", Gurrutxaga et al. A&amp;A, 682, A43 (2024)


disc.py and protoplanet.py constain all equations. Scripts for plotting are added by figure.
